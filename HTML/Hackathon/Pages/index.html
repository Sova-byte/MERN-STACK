Before we look at the code used to build websites it is important to consider the

üî¥ BROWSERS

üí† People access websites using software called a web browser. Popular examples include Chrome, Brave, Safari, Microsoft Edge, and Opera.

üí† When you ask your browser for a web page, the request is sent across the Internet to a special computer known as a web server which hosts the website.

üí† Web servers are special computers that are constantly connected to the Internet, and are optimized to send web pages out to people who request them.

üí† Some big companies run their own web servers, but it is more common to use the services of a web hosting company who charge a fee to host your site.

üí†People are accessing websites on an increasing range of devices including desktop computers, laptops, tablets, and mobile phones. It is important to remember that various devices have different screen sizes and some have faster connections to the web than others

All websites use HTML and CSS, but content management systems, blogging software,
and e-commerce platforms often add a few more technologies into the mix.

üî¥ WHAT YOU SEE

üí† When you are looking at a website, it is most likely that your browser will be receiving HTML and CSS from the web server that hosts the site. The web browser interprets the HTML and CSS code to create the page that you see.

üí† Most web pages also include extra content such as images, audio, video, or animations.

üí† Some sites also send JavaScript to your browser, and you will see how to add JavaScript and Flash in your web pages. Both of these technologies are advanced topics that you can go on to learn more about once you have mastered HTML and CSS, if you want to.

üî¥ HOW IT IS CREATED

üí† Small websites are often written just using HTML and CSS.

üí† Larger websites ‚Äî in particular those that are updated regularly and use a content management system (CMS), blogging tools, or e-commerce software ‚Äî often make use of more complex technologies on the web server, but these technologies are actually used to produce HTML and CSS that is then sent to the browser. üí† Larger, more complex sites like these may use a database to store data, and programming languages such as MySQL, MongoDB, Java, or ExpressJS on the web server, but you do not need to know these technologies to improve what the user sees. üî¥ HTML5 & CSS3
